## Quer rodar localmente e não sabe como? Vem que eu te ensino.

Clone o projeto

```bash
  git clone https://github.com/ianaugustosilva/date-me
```

Entre no diretório do projeto

```bash
  cd date-me
```

Instale as dependências

```bash
  yarn install
```

Inicie o servidor

```bash
  yarn start
```
